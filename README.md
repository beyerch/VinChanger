# VinChanger
WAY before car hacking was cool, some of us we out there hacking away. This is a VB6 utility created to change VIN #'s in GM Vehicles via $3B/$3C block operations.  Worked with the B&amp;B Electronics Autotap cable and later the DHP interface.
